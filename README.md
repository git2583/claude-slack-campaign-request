## 🔗 Slack 연동 — 자동 입력 기능

Slack `#campaign-request` 채널에 올라온 캠페인 요청 메시지를 읽어  
HTML 입력 폼에 자동으로 값을 채워 넣습니다.

### 실제 적용 예시 (2026-04-19)

```
[캠페인 요청] 장바구니 이탈 고객 "48시간 한정 전환 이벤트 세팅 요청"

1) 캠페인명: 장바구니 이탈 48시간 한정 쿠폰 (4월 1주)
2) 목표: 구매 전환(결제 완료)
3) 대상: 최근 14일 내 장바구니 담기 후 미구매 유저
   (제외: 최근 7일 내 구매 완료/환불 경험 유저)
4) 혜택: 10% 할인 쿠폰(최대 10,000원), 48시간 유효, 최소 30,000원
5) 채널: 이메일/카카오(1순위) → 메타 리타겟(2순위) → 구글(3순위)
6) 예산: 총 2,500,000원 (Meta 1,700,000 / Google 800,000)
7) 일정: 세팅 완료 4/2(목) 오전 / 집행 4/3(금)~4/9(목)
8) 랜딩 URL: 미정
9) 핵심 메시지: "장바구니에 담아뒀던 상품 48시간 안에만 10% 할인으로 마무리하세요"
```

**자동 입력 처리 방식**
- `DOMContentLoaded` 이벤트 기반 prefill 스크립트로 11개 필드 자동 주입
- 채널 체크박스 자동 선택 (이메일 · 메타 · 구글)
- 우측 상단 완료 배너 5초 노출 후 자동 소멸
